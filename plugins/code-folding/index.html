Code Folding

Code Folding specific line ranges.

How to use

Obviously, this is supposed to work only for code blocks (<pre><code>) and not for inline code.

You specify the lines to be highlighted through the data-folding-open attribute for an open folding or the data-folding-closed attribute for an closed folding on the <pre> element, in the following simple format:

  • Ranges are denoted by two numbers, separated with a hyphen (-)
  • Multiple line numbers or ranges are separated by commas.
  • Whitespace is allowed anywhere and will be stripped off.

Examples:

1-5
Lines 1 through 5
1-2, 9-20
Lines 1 through 2, lines 9 through 20
